# MythRealms Virtual Talent V1

## Purpose

Reference library for short TikTok product videos. This is a fictional adult
brand model inspired by the supplied reference's summer editorial mood, not a
replica of the person in that image.

## Character Lock

- Adult Mediterranean-European woman, age 29
- Warm fair skin with visible freckles and hazel eyes
- Dark brown, softly wet-textured bob
- Understated makeup and natural skin texture
- Warm direct summer sunlight with organic leaf shadows

## Reference Order For Video Generation

1. Upload one identity image that matches the intended camera angle.
2. Upload the relevant product's clean canonical image and an already-approved
   on-model keyframe for that SKU when available.
3. Upload one scene plate and one wardrobe asset.
4. Generate a single 4-6 second action only. Use quiet motions: turn toward
   light, tuck hair behind the ear, adjust a collar, or rotate the wrist.
