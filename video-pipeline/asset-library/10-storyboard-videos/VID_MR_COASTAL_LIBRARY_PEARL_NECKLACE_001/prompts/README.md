# Seedance 提示词使用方式

所有提示词均为 16:9、全能模式、至少 4 秒生成。每条文件开头列出的 Windows 绝对路径都应作为参考图上传。

- S01–S04、S07、S09 是独立开镜。
- S05 生成后必须导出验收末帧，保存为 first-frames\accepted-s05-last-frame.png。
- S06 的提示词中，将该文件作为第一张输入图；验收 S06 后同样保存 accepted-s06-last-frame.png。
- S08 用 accepted-s06-last-frame.png 续接。若末帧与首帧不一致，实际验收末帧优先。
- 提示词不要求生成文字；声音仅为节奏与剪辑参考，可在后期统一。
