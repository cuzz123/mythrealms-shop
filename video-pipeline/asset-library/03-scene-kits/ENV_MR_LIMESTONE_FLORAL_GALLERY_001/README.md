这是一组同一空间的世界锚点，只控制建筑、石材、晨光方向、植物投影与展台位置；不控制人物身份、首饰结构或产品摆放。

## 空间连续性

- 材质：未经抛光的浅灰白石灰岩，允许细小孔隙、接缝和天然颗粒。
- 光向：高窗外的晨间自然光由画面左侧进入；不得替换为冷蓝光或棚拍聚光。
- 花影：室外树叶在地面与墙面形成缓慢、细碎的投影；视频中仅以微小漂移表现。
- 主展台：画面中心或低前景的空石灰岩台，S01–S05 保持为空，不作为首饰替代品。
- 侧展台：远处白色石膏花雕只承担空间节奏；其粗粝花形与耳环的透明花瓣不相同。

## 禁止转移

无人、无首饰、无产品、无包装、无品牌标识、无可读文字、无黑色天鹅绒、无博物馆标签、无人工彩色灯光。
