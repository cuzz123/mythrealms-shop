# ENV_MR_NATURAL_WINDOW_LINEN_APARTMENT_001

真实自然光窗边亚麻公寓：Pearl Series 04 重构的环境锚点。

## 文件

- `source/window-apartment-panorama-v1.png`：从室内朝窗的主全景。窗、椅子、木桌和人物活动区的空间依据。
- `source/window-apartment-reverse-v1.png`：从窗边朝室内的反向角度。用于人物转身、从窗边向室内的反拍与剪辑连续性。

## 连续性规则

- 保持奶油矿物墙、象牙色亚麻帘、燕麦色亚麻椅、浅胡桃木、浅橡木地板和柔和午前窗光。
- 人物从椅子走向窗的移动，必须沿两张图中保留的中间通道完成。
- 该场景只承担生活方式空间；不得加入黑色舞台、奇幻烟雾、古董匣或强聚光。
