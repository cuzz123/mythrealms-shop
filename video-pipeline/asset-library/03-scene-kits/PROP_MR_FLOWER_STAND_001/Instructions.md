# 花艺陈列架

木质矮架、三只金属花桶和浅色玫瑰组成的静态场景道具。适合花店、露台和生活方式镜头的前景层次。

## 使用

- 使用 `MR_FLOWER_STAND.blend` 作为预览或 Link 源。
- 原始 GLB 位于 `source/hunyuan-web/MR_FLOWER_STAND.glb`。
- 道具整体可缩放、旋转和移动；需要换花材时，需先拆分并重做材质归属。

## 验收边界

- 已验收：完整架体、花桶、花束与基础 PBR 预览。
- 未验收：可替换花材、花朵摆动、近景微距花瓣精度。
