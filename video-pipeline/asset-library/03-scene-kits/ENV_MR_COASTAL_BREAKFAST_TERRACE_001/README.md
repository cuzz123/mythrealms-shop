# ENV_MR_COASTAL_BREAKFAST_TERRACE_001｜海岸早餐露台

真实地中海住宅露台的环境锚点，用于「来自海岸的一封信」项链视频。

## 资产文件

- `source/coastal-breakfast-panorama-v1.png`：面向远海的建立镜头；锁定低矮石灰岩海墙、浅木圆桌、白亚麻、白瓷碟、整颗柠檬与晨间远海。
- `source/coastal-breakfast-reverse-v1.png`：从餐桌望向石屋的反打；锁定门洞、白亚麻帘、石材、桌面和相同的侧向晨光。

## 连续性规则

- 石材为偏暖的米白石灰岩；桌面为浅色原木；亚麻为哑光米白。
- 海面保持低饱和蓝绿色，海平线低且有晨雾；不出现度假酒店、泳池、旅游海报色彩。
- 两图中的白瓷碟在 S01–S05 必须为空，S06 才可放入产品。
- 环境图不控制人物身份，也不控制项链结构。

## 禁止元素

黑金奇幻布景、过度干净的 CGI 材质、可读信件文字、品牌标志、水印、人物与首饰。
