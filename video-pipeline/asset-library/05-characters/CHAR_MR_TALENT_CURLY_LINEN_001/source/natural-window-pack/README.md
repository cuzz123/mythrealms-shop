# Natural Window Talent Pack

`CHAR_MR_TALENT_CURLY_LINEN_001` 的自然窗边扩展资产。用于真实生活方式片，而非礼服或舞台化广告。

## 身份锁

- 成年女性；暖中性肤色、短深色自然卷发、轻刘海、低妆感。
- 服装固定为象牙白亚麻连衣裙；皮肤保留真实毛孔与自然不对称。
- 禁止改发色、加重妆、塑料磨皮，或变成 3D / 时装秀人物。

## 文件

- `turnaround-v1.png`：正面、侧面、背面身份和服装锁。
- `expressions-v1.png`：自然微表情（阅读、望窗、低头、呼吸、轻微回应）。
- `motion-contacts-v1.png`：坐读、起身、走向窗、碰窗帘、整理袖口、触耳侧的动作接触关系。

## 使用规则

- 产品出现前可使用全部动作；产品出现后，以产品结构锁图为准，手部不得遮挡核心钴蓝石与粉珍珠簇。
- 近景优先引用 `expressions-v1.png`，全身 / 走位优先引用 `motion-contacts-v1.png`。
