# CHAR_MR_TALENT_LIMESTONE_GALLERY_001｜石灰岩花影馆模特

## 身份锁

真实成年女性、深棕自然微卷短发、自然肤质、奶油白无袖长裙。她在本包中只承担人物身份与无首饰的动作基础；耳环必须由产品锁在 S06 以后单独控制。

| 文件 | 单一职责 |
| --- | --- |
| `source/turnaround-v1.png` | 正面、左前侧和背面的人物比例、脸型、发长与服装锁。 |
| `source/expressions-v1.png` | 克制中性、侧向凝视、垂眸、安静半侧脸；不包含首饰。 |
| `source/motion-contacts-v1.png` | 走入、侧身停下、慢侧头、微风带动少量发丝的动作节奏；不作为耳环图。 |

## 使用约束

- S01–S05 的耳朵必须保持裸露或被发丝遮挡；禁止生成耳环、项链、戒指、手链。
- 不直视镜头、不笑、不做夸张表演；动作应以轻微呼吸、停顿和少量发丝位移完成。
- 所有最终首饰画面另行引用 `PROD_MR_WHITE_SHELL_FLOWER_DROPS_001` 的产品结构源图。
