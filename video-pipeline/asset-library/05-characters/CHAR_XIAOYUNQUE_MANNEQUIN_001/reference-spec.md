# 小云雀导演台假人：四视图重建规范

> 013 实现说明：当前母资产采用自然比例连续表面作为核心，蛋形头替换人脸，并以同表面材质分区表示关节；与早期“全部刚性独立模块”规范不同，主体现为已自动权重绑定的 55 骨骼网格。参考三分之四轮廓 IoU 已验证为 62.27%。

## 视觉结构

- 成熟女性比例：头高约为全高的 1/7.5；肩宽约 2.2 个头宽；髋部略宽于肩部。
- 无五官椭圆头；细颈连接独立胸腔；胸腔、细腰、骨盆连续且有硬表面切线。
- 肩、肘、腕、膝、踝可见圆柱/环状机械关节；四肢有纤细渐缩的连续壳体。
- 手掌是扁平楔形；五指均为三段指骨并具有指关节间隙。
- T-pose，手臂仅轻微下垂；脚尖朝前，双腿自然分开。

## 机械骨骼与绑定（可验证的目标）

每个可见模块刚性绑定到其单一骨骼：root、pelvis、spine_01、spine_02、neck、head、clavicle、upper_arm、forearm、hand、每根手指 3 节、thigh、shin、foot、toe。关节独立对象绑定到相邻的父骨骼。该方案匹配可见的机械假人运动逻辑，但不能声称与未提供的原始隐藏骨骼完全相同。

## 评分门槛

- 正面、侧面、背面分别评分，剔除 UI 标签、网格和坐标轴。
- 首要评分：遮罩轮廓 IoU；辅助评分：关键关节、手指端点和身体比例误差。
- 仅当三个可见视图均 >= 90% 时，才可称为“视觉 90% 还原”。
