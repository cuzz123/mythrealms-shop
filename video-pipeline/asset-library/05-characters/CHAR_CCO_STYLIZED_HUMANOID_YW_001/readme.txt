Young Warrior (Rigged Stylized Humanoid Character Base) — [CC0 license]

This is a stylized humanoid character base model intended for use in games, prototypes, animation tests, RPG projects, visual novels, and other real-time 3D projects.

The model includes a skeletal rig / bone animation setup, making it suitable for basic character animation, posing, and further customization. It can be used as a base mesh for creating NPCs, player characters, fantasy characters, or custom humanoid variants.

The character has a slim stylized body, anime-inspired proportions, simple facial features, and a clean game-ready look. The model is provided as a free asset for developers, artists, students, and hobbyists.

Features
Stylized humanoid character model
Rigged with bones / skeletal animation
Textured (color + additional layer of details)
Suitable for game engines
Can be used as a base character mesh
Editable and customizable
