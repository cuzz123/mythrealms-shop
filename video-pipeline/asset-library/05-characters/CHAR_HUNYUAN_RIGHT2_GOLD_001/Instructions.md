# 右二｜混元 3D 金色礼服角色

由右二三视图生成的 PBR 角色：金色礼服、耳饰、手链和盘发已经成为模型的一部分。它适合与 C 位组成双人对位、正反打和车边中近景。

## 调用

1. 在 Blender 打开 `RIGHT2_GOLD.blend`，或导入 `source/hunyuan-raw/RIGHT2_GOLD.glb`。
2. 保留 Hunyuan 原始 PBR 材质，再按镜头需求校准色温、反射和高光。
3. 当前模型无骨骼，且下半身到膝部附近收口；不能直接用于走路、全身落地或多人全景。
4. 要进入动作库，先补全下肢/鞋履网格，再重拓扑和绑定到 [[CHAR_HUMAN_MASTER_RIG_001｜人体标准骨架]]。

## 验收

- 生成模式：Hunyuan 3D Pro 3.1，Normal + PBR。
- 输入：正面、右前 45°、背面三视图。
- 已在 Blender 5.1.2 成功导入、渲染 `Thumbnail.png`。

## 使用边界

可用于中近景角色预演、金属礼服灯光、双人构图和侧身回眸；暂不用于全身走位、复杂肢体动作或最终商业角色交付。对外商用前仍需确认参考图来源权利与腾讯混元服务条款。
