# STYLE_VIEW_POSITION_MEDIUM_001

参考文章提炼的“观看位置 / 媒介质感”决策系统。它不是一个滤镜，也不是把多个设备词堆在一起；先决定观众从哪里看，再决定为什么要让观众这样看。

## 调用顺序

1. 先选时间距离：手持、固定机位、长镜头或慢动作。
2. 再选透视关系：俯拍、荷兰角、窥视、广角、长焦或微距。
3. 若需要画面具有记录来源，再只选一种媒介：GoPro、FPV 或 Bodycam。
4. 若需要一个信息记忆点，再只选一种表达设备：红外、热成像、X 光、玻璃反射或鱼眼。

## 与现有资产的组合

- 产品亲密感：`CAM_SLOW_PUSH_MACRO_85MM_001` + 微距 + 固定/缓慢推进。
- 人物珠宝揭示：`CAM_ORBIT_EAR_REVEAL_50MM_001` + 长焦或玻璃反射，不叠加鱼眼。
- 群像移动：`CAM_MULTI_MODEL_SLIDE_50MM_001` + 长镜头；FPV 仅在主观追随有叙事理由时使用。

## 边界

- 一条短片最多使用一种“记录设备感”和一种“分析设备感”。
- 设备词必须改变观看关系或叙事信息；不能只为视觉奇观。
- 本资产只保存可复用方法，不复制原文图像、视频或提示词。
