# STYLE_AUTHORIAL_MOOD_MATRIX_001

参考文章所说的“导演与作者气质”。这里把引用到的创作特征转成可执行的通用情绪系统，避免把创作者姓名当作万能提示词。

## 使用方法

1. 只选一个主情绪系统，写清它要服务的故事信息。
2. 从其中选 2–4 个关键词，并落实到构图、镜头速度、色温、表演和声音。
3. 若需要第二系统，只允许作为最后 20% 的转场或反差，不与主系统平均混合。

## 影视资产库中的优先组合

- 奢华珠宝：`ICONIC_COLD_ORDER` + `LIGHT_MOON_GOLD_001` + 慢推 / 微距。
- 都市人物：`ROMANTIC_URBAN_MEMORY` + 横向滑轨 + 玻璃反射。
- 悬疑短剧：`SUSPENSE_INFORMATION_GAP` + 前景遮挡 + 拉焦。
- 假人排练：`PSYCHOLOGICAL_PRECISION` + `ACT_HEAD_TURN_SLOW_001`，先检查视线后再做大动作。

## 来源与边界

来源文章提到多位创作者与动画工作室作为气质案例；本资产只存储抽象后的视觉方法，不存储或要求复刻任何具体作品、角色、品牌或标志性镜头组合。
