# 标签微距贴近

约 2.50 秒。85mm 微距视角推断，硬切到包装标签或材质细节，保持稳定的近距离贴近，不依赖明显摇镜。

## 相机契约

- 景别：特写到大特写
- 机位：正前方略低角度
- 路径：极短距离微推，允许少量横向修正
- 速度：慢且稳定
- 焦点：锁定标签、刻字或主材质纹理
- 终点：细节占据画面主体，但仍能看出产品曲面

## 复用

适合读标、看材质、珠宝纹理和金属刻字。对焦面很薄，避免同时移动产品、相机和焦点。
