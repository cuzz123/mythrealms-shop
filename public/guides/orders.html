订单处理 + 发货 + GMC 操作指南

一、每日: 处理新订单

1.1 查看订单

  1. 打开 https://mythrealms-shop.vercel.app/admin/orders
  2. 登录: admin@mythrealms.com / admin123
  3. 看状态列: PENDING (待付款) / PAID (已付款待发货) / SHIPPED / DELIVERED

1.2 发货流程

步骤操作
1. 确认付款检查 LS 或 PayPal 后台确认款项已到账
2. 安排制作手工制作需要 2-3 周，通知供应商
3. 发货拿到运单号后 → Admin Orders → 点 "Ship" → 状态变为 SHIPPED
4. 通知客户运单号填入订单备注
5. 确认送达物流显示签收后 → 点 "Deliver" → 状态变为 DELIVERED

1.3 LemonSqueezy 后台

  1. 打开 app.lemonsqueezy.com/orders
  2. 检查新付款 → 确认金额正确 → 在 Admin Orders 中标记为 PAID

1.4 PayPal 后台

  1. 打开 sandbox.paypal.com (沙盒) 或 paypal.com (正式)
  2. 检查交易记录 → 确认款项状态

二、每周: 检查库存

  1. Admin → Products → 查看所有产品
  2. 筛选库存 < 5 的产品
  3. 如果某个产品卖出较多 → 补货或调整库存数字
库存扣减在 Webhook 触发时自动执行。退款时库存自动回补。

三、GMC (Google Merchant Center) 检查

3.1 每日检查

  1. 打开 merchants.google.com
  2. 左侧 → "产品" → 看概览页面
  3. "未核准" 数量是否在减少？
  4. 点击未核准产品 → 查看详细拒绝原因

3.2 常见拒绝原因及修复

拒绝原因修复方法
缺少 GTIN✅ Feed 已声明 identifier_exists=no
图片不符合要求检查产品图是否清晰、白色背景、无水印
缺少运费信息✅ 已在 GMC 设置中配置
价格不一致确认网站价格和 Feed 价格一致
缺少退货政策✅ /refund 页面已有完整内容

3.3 Feed 状态

左侧 → 产品 → Feeds → 点击 MythRealms Products → 看 "上次提取" 时间。应该每天自动更新。

如果显示 "失败" → 检查 Feed URL: https://mythrealms-shop.vercel.app/api/feed/google 是否可访问。

四、Canva 做图指南

4.1 Pinterest 竖版 Pin (1000x1500)

  1. 打开 canva.com
  2. Create a design → 搜索 "Pinterest Pin" → 选 1000x1500
  3. 左侧 Uploads → 上传产品图
  4. 拖产品图到画布中间偏上 (占 60%)
  5. 下方空白加文字:
    • 第一行: 产品英文名 (Add a text box → Serif 字体 → #D4A84B → 字号 36-48)
    • 第二行: 材质+价格 (Sans-serif → 白色 → 字号 24)
  6. Background 改成 #0F0D0E
  7. Share → Download → PNG → 上传到 Pinterest

4.2 关于文字叠加

Pinterest 推荐图片上有 2-8 个词的文字叠加，说明这是什么东西。例如: "14k Gold Pendant" / "Handcrafted Jade Bracelet"。

文字区不应超过图片面积的 20%。
