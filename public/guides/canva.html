Canva 做图指南

Pinterest 竖版 Pin (1000x1500 px)

  1. 打开 canva.com → 免费注册
  2. Create a design → 搜索 "Pinterest Pin" → 选 1000x1500
  3. 左侧 Uploads → 上传产品图 (从 public/images/products/)
  4. 拖产品图到画布中间偏上 (占 60%)
  5. Add a text box:
    • 第一行: 产品英文名 (Serif 字体, #D4A84B, 字号 36-48)
    • 第二行: 材质+价格 (Sans-serif, 白色, 字号 24)
  6. Background 颜色改成 #0F0D0E
  7. Share → Download → PNG → 上传到 Pinterest

文字叠加不超过图片 20%。2-8 个词最佳。
