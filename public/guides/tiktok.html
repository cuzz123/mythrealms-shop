TikTok 操作指南 (2026年6月)

IP 稳定性 (最重要)

  1. 打开 whatismyipaddress.com
  2. 记录当前 IP 地址
  3. 2 小时后、6 小时后再查一次
  4. 如果 IP 变了 → 不要发内容，等 IP 稳定
IP 频繁切换 + 发内容 = 秒封。浏览不打紧，发视频必须 IP 固定。

每日: 养号 (第 1-14 天)

浏览 15 分钟: 正常刷 For You → 遇到感兴趣的视频完整看完 → 点赞/收藏 → 遇到不感兴趣的等 3-4 秒再划走。

搜索: 在搜索框输入 jewelry collection / mythology / crystal healing / luxury lifestyle → 浏览结果。

前 3 天: 不发任何内容，纯浏览。

第 4-7 天: 关注 10-15 个同品类账号 → 评论 5-8 条 → 分享 2-3 条。

发布视频

  1. 用手机系统相机拍摄 (不用 TikTok 相机，画质更好)
  2. 产品慢速平移特写: 15-20 秒，不说话
  3. 配古风音乐: TikTok 音效库搜索 "ancient Chinese instrumental"
  4. 加字幕: 用 TikTok 内置文字工具 (不要用外部 App)
  5. 描述: 2-3 句英文 + 3-5 个标签
  6. 发布后: 自己完整看一遍 + 给自己点赞 + 回复前 3 条评论
不要: 搬运视频 / AI 语音朗读 / 每条带链接 / 频繁切换 IP
