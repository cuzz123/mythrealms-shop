# MythRealms「珍珠潮汐谷」首段分镜设计

日期：2026-07-17  
项目：参考 `9cac57b8c711eb815d9bc0d15c01a29c.mp4` 的广告结构，为 MythRealms 制作原创珍珠品牌片  
范围：仅制作第一段“陌生风景与生活幻觉”，不提前进入人物、野马、奔跑、产品特写或品牌结尾

## 1. 首段任务

第一段存在的唯一理由，是让观众相信画面通向一种现实中难以抵达的生活。它先建立世界，不解释产品。

旁白：

> 一条高级的广告，一开始都必须从大众未见过的风景开始。卖什么不重要，但得制造一种人们想要的生活幻觉。

首段终点：镜头停在海崖宅邸的暖色窗光上；下一段才进入“看似拥有一切的女人”。

## 2. 原创性与参考边界

- 原视频只控制功能与节奏：遥远自然空间 → 更深的空间穿行 → 孤立建筑揭示。
- 不复制原片的具体岛屿、峡谷、建筑、人物、字幕、音乐、Logo 或镜头逐帧构图。
- MythRealms 的原创世界由黑色玄武岩、液态银潮水、贝母虹彩岩层和黑曜石玻璃宅邸构成。
- 现有参考画板只控制版式与信息密度，不复制其中模特、场景或首饰。

## 3. 视觉圣经

### 世界

- 黎明前的远海群岛，岛体为湿润黑色玄武岩。
- 海水呈低饱和银灰色，不能像水银或科幻液态金属；仍须保留真实海浪和泡沫物理。
- 峡谷岩壁存在天然贝母薄层，随视角出现淡粉、淡金、冷青三色虹彩，不形成巨大珍珠或直白首饰造型。
- 悬崖尽头是一座低矮、水平延伸的黑曜石与无框玻璃宅邸；建筑可居住、克制、没有科幻飞船轮廓。

### 色彩与光线

- 主色：海雾蓝、玄武岩黑、珍珠银。
- 辅色：贝母淡粉和极少量暖金。
- 稳定冷色阴天环境光；唯一暖色来自宅邸内部一扇窗。
- 不使用炫光泛滥、紫色霓虹、魔法粒子、极光或高饱和天空。

### 导演声音

- 采用“克制的构图古典主义”：平衡构图、缓慢且有目的的移动、长呼吸、低密度声场。
- 镜头尺度由极远景逐步收紧，运动由高空漂移转为贴近地形，最后减速并静止。
- 第一段不出现人物，因此情绪由地理尺度、光线差异和建筑位置承担。

## 4. 三镜头结构

### S01A 世界锚点｜0–4 秒

- 画面：极远景俯瞰黑色玄武岩群岛，低云压住山脊，银灰海水穿过岛屿。
- 镜头：24mm 航拍感，缓慢向前并轻微下降；不旋转、不俯冲。
- 终点：中央裂谷成为清晰的前进方向。
- 光线：冷色天光，云隙只在远处海面形成一条窄亮带。
- 声音：低风、远浪，无音乐旋律。

### S01B 峡谷穿行｜4–8 秒

- 画面：35mm 低空进入裂谷，左右湿岩壁出现受控贝母虹彩，海水在下方真实拍岸。
- 镜头：沿峡谷单向前移，保持水平；不摇晃、不贴岩急转。
- 终点：峡谷出口出现一个极小的几何建筑剪影。
- 光线：冷色环境光；贝母层只在掠射角度短暂显色。
- 声音：风声收窄、浪声靠近，加入一次低频脉冲。

### S01C 宅邸揭示｜8–12 秒

- 画面：50mm 中远景揭示悬崖上的黑曜石玻璃宅邸，远海与云墙在背景；建筑内部只有一扇暖窗亮起。
- 镜头：从岩石遮挡后缓慢侧移揭示，随后减速到静止。
- 终点：暖窗位于右上三分点，作为下一段人物栖居场景的入口。
- 光线：稳定冷天光与单一暖窗形成冷暖对比，不让整栋建筑发光。
- 声音：风声被建筑挡住后变闷，旁白结束，保留半秒安静。

## 5. 画板交付

采用用户参考图二的横向“总导演画板”结构，但只覆盖第一段：

- 顶部：`MYTHREALMS / PEARL TIDAL VALLEY / OPENING 0–12s`。
- 主体：三张 16:9 首帧，依次为 S01A、S01B、S01C。
- 每格下方：时间、画面任务、运镜、焦点、光线、声音、旁白。
- 底部：世界观材质条、色板、镜头递进图和 Seedance 风险标签。
- 画板文字由后期排版完成，不要求图像模型生成可读中文。

同时生成四张无字引用图：

1. `world-anchor`：群岛超远景，控制地理与色彩。
2. `ravine-anchor`：峡谷正向构图，控制运动通道和贝母岩层。
3. `villa-anchor`：宅邸三分之四远景，控制建筑身份与暖窗位置。
4. `material-anchor`：玄武岩、海水泡沫、贝母薄层的近景材质板。

## 6. Seedance 2.0 可控性设计

这一段属于高场景密度、低角色密度。主要预算投入世界与建筑连续性，不投入人物身份或产品微距。

### 风险

- 单次生成同时要求群岛、峡谷穿行和宅邸揭示，容易造成地形跳变与建筑变形。
- 复杂航拍词叠加会造成镜头旋转、俯冲或速度突变。
- “贝母岩层”可能被误生成为巨大珍珠、贝壳宫殿或魔法发光材质。
- 宅邸在运动中可能改变楼层、窗户数量和轮廓。

### 处理

- 将首段拆为三个独立 I2V 片段，每段一个可见动作和一个镜头终点。
- S01A 使用 `world-anchor`；S01B 使用 `ravine-anchor`；S01C 同时使用 `villa-anchor` 与 `material-anchor`，各自只承担一个角色。
- 每段只写一个主镜头动作；不把无人机上升、下降、环绕和推进堆在一起。
- 建筑揭示段锁定宅邸轮廓、楼层、窗户位置和暖窗位置，只允许云、海浪、雾和相机变化。
- 三段分别生成并在剪映中剪辑；音乐统一放到后期，生成阶段只保留环境声意图。

## 7. 连续性锁定

- 时间始终为同一黎明前时段。
- 海水、云层、主风向和冷暖关系不重置。
- 运动方向始终向岛屿深处推进。
- 贝母虹彩仅为岩层薄片，不升级为新物体。
- S01C 的宅邸暖窗是第二段人物出场的空间锚点。
- 第一段禁止出现模特、马、首饰特写、产品包装、文字或 Logo。

## 8. 验收标准

1. 三张首帧能在不看说明时形成“由远到近、进入某种生活”的连续阅读。
2. 场景陌生但可信，不像游戏地图或科幻概念飞船。
3. 贝母暗示珍珠品牌，但不提前泄露产品。
4. 每个视频提示词只有一个主镜头动作和明确终点。
5. 宅邸造型在引用图和生成视频中保持一致。
6. 画板可直接用于小云雀或 Seedance 上传工作流，并保留无字原图供后续引用。
