# BuddhaStonesShop vs MythRealms — 首页设计对比评估

## 一、核心差异

| 维度 | BuddhaStonesShop ($800万/年) | MythRealms |
|------|------|------|
| 分类逻辑 | **情绪索引** (Protection / Anxiety Relief / Energy) | 产品类型 (Beasts / Mansions / Zodiac) |
| 产品卡片 | 场景佩戴图 + 情感文案 + 价格 | 产品图 + 名称 + 材质 |
| 轮播图 | 满屏 (100vh) + 左右滑动 | ✅ 刚改完 (85vh + 水平切换) |
| 首页节奏 | 轮播 → 分类 → 产品 → 产品 → 产品 | 轮播 → 分类 → 产品 → 28宿 → Guardian → Newsletter |
| 社交证明 | UGC 照片墙 + 社媒嵌入 | 无 |
| 信任条 | 免邮/退货/安全支付 (窄条常驻) | 散落在各处 |
| 底部 | 政策 + 社媒 + 邮件订阅 | 有 |

## 二、我们要改的 5 件事

### 1. 分类改名 → 情绪化
改前: Mythical Beasts / 28 Mansions / Zodiac Amulets / Four Seasons
改后: Find Your Guardian / Find Your Star / Find Your Destiny / Find Your Season

### 2. 加信任条 (Hero 下方常驻窄条)
"Free Shipping Over $69.99 | 30-Day Returns | Handcrafted to Order | Secure Checkout"

### 3. 产品卡片加情感标签
每张卡片加一行小字: "For the one who..." （跟产品页 Guardian Tag 呼应）

### 4. 加社交证明区
即便是模拟数据，也要有 "Join 1,000+ Guardians" + 几张产品佩戴图排列

### 5. Footer 加强
加：支付图标行 / 社媒图标大号 / Newsletter 订阅
