# MythRealms TikTok 运营完整操作手册 (2026年6月最新版)

---

## 前提: IP 问题

TikTok 对 IP 一致性极其敏感。如果你的 VPN 频繁换 IP，**暂时不要做 TikTok**，先做 Pinterest。

如果要做 TikTok，先用以下方法测试你的 IP:

### IP 稳定性测试

1. 打开 `whatismyipaddress.com`，记录下当前 IP
2. 过 2 小时再看一次
3. 再过 6 小时再看一次
4. 如果三次 IP 都一样 → 可以用
5. 如果 IP 变了 → 不要做 TikTok，等用了固定 IP 再说

## 第一部分: 注册与设置 (前提: 已解决 IP 问题)

### 1.1 手机准备

**方案一: 买专用手机**
- 淘宝搜 "TikTok 运营手机"
- 300-500 元，预装 TikTok + 固定节点
- 插电即用，不用折腾

**方案二: 用自己手机**
- 必须是**闲置手机**，不能是日常用的手机
- iPhone: 需要海外 Apple ID
- Android: 需要 Google Play + 谷歌服务框架
- 手机语言改成 **English (US)**
- 时区改成 **New York (EST)**
- 定位**关闭**
- **取出 SIM 卡** (重要! 国内 SIM 卡插着会被检测)

### 1.2 注册 TikTok

1. 连接 VPN，确认 IP 稳定
2. 打开 TikTok App
3. 点 **"Sign Up"**
4. 选 **"Use Email"** (不要用手机号! 国内手机号会触发风控)
5. 输入 Gmail 或 Outlook 邮箱
6. 设置密码 (12 位以上)
7. 填写生日 (成年)
8. 完成验证码 (可能需要邮箱验证)

**注册后不要立即改名和换头像** — 先浏览 2-3 天。

## 第二部分: 养号期 (第 1-14 天)

### 2.1 第 1-3 天: 纯浏览

**目标: 让 TikTok 算法把你标记为正常美国用户。**

**每天操作 (15-20 分钟):**

1. 打开 TikTok，浏览 **For You** 推荐页
2. 每看 3-5 个视频，停顿 2-3 秒再划下一个 (不要秒划)
3. 遇到感兴趣的视频:
   - **完整看完** (等进度条走完)
   - **点赞** (点心形)
   - **偶尔收藏** (点书签图标)
4. 遇到不感兴趣的视频:
   - **正常划走** (不要秒划，等 3-4 秒再划)
5. 不要做: 评论、关注、搜索、发视频

**搜什么内容 (第 3 天开始):**

在搜索框输入以下关键词，浏览搜索结果:
- `jewelry collection`
- `mythology`
- `crystal healing`
- `luxury lifestyle`
- `ancient history`
- `witchtok`

> 这告诉 TikTok 你的账号属于哪个 "内容池"。搜什么，TikTok 就把你推给同类用户。

### 2.2 第 4-7 天: 开始互动

**在 2.1 的基础上增加:**

1. **关注 10-15 个账号**:
   - 珠宝类: 搜 "jewelry brand"、"handmade jewelry"，关注中小账号
   - 神话类: 搜 "mythology"、"ancient history"，关注讲神话的账号
   - 水晶类: 搜 "crystal collection"、"crystal healing"，关注水晶博主
   - 奢侈品类: 搜 "luxury unboxing"、"designer jewelry"

2. **评论 5-8 条**:
   - 在珠宝类视频下评论 (简单如 "Love this!" "So elegant" "Where is this from?")
   - 不要发广告、不要提 MythRealms、不要放链接

3. **分享 2-3 条**:
   - 点分享按钮 → 复制链接 → 不一定要真的发给别人，复制到剪贴板就行
   - TikTok 只看你有没有点分享，不看真的分享到哪

**每天时间分配:**
- 浏览: 10 分钟
- 互动 (点赞/收藏/评论/分享): 5 分钟
- 关注: 2 分钟
- **不发任何视频**

### 2.3 第 8-14 天: 开始发内容

**第一条视频 (第 8 天):**

**拍什么:**
1. 找一个自然光好的窗边
2. 在白色背景 (白纸、白桌布、白墙) 上
3. 放一件 MythRealms 产品
4. 手机竖着拍，**慢速平移** (从左到右移动手机，展示产品)
5. 拍 15 秒
6. 不要说话，不要配乐 (用手机原声)

**怎么发:**
1. 点底部中间 **+** 号
2. 从相册选择你拍好的视频
3. 点 **"Next"**
4. **不要加任何特效、滤镜、字幕** (第一条视频越自然越好)
5. 描述 (Caption) 填写: `morning light`
6. **不要放任何标签 (#hashtag)**
7. **不要放链接**
8. 点 **"Post"**

**发布后 (重要):**
1. 自己完整看一遍
2. 给自己点赞
3. 发布后 1 小时内不要频繁刷新看播放量
4. 1 小时后看播放量:
   - 200-500+ → 账号健康，继续
   - 50-200 → 可能部分限流，再看下一条
   - 0-50 → 已被标记，放弃这个号

**第 9-14 天，每天 1 条:**

| 天 | 内容 | 时长 | 说明 |
|------|------|:--:|------|
| 9 | 另一件产品的平移特写 | 15s | 验证是否被限流 |
| 11 | 手持产品，自然光下展示 | 15s | 有手入镜 = 原创度高 |
| 13 | 产品 + 搭配场景 (丝绸/大理石/书) | 15s | 展示 "生活方式" |

> 前两周只发生活化内容，简介不要挂链接，不要提 "MythRealms"。先让算法接受你。

## 第三部分: 正式运营 (第 15 天起)

### 3.1 更新账号资料

1. 头像: 上传 MythRealms logo 或一张产品图
2. 昵称: `MythRealms`
3. 简介 (Bio):
   ```
   Ancient Chinese mythology meets modern luxury jewelry.
   🦊 Nine-Tailed Fox · Phoenix · 28 Celestial Guardians
   Find your guardian ↓
   ```
4. 简介里的链接: 填写你的网站 `mythrealms-shop.vercel.app`

> 简介链接第 15 天才加！前面两周不要放。

### 3.2 内容矩阵 (每周 3-4 条)

**类型 1: 产品美学 (50%)**

- 15-20 秒
- 产品特写: 慢速平移/旋转/逆光
- 配古风音乐 (在 TikTok 音效库搜索 "ancient Chinese"、"instrumental")
- 不说话
- 字幕叠加: 产品名 + 材质 + 价格 (用 TikTok 自带的文字工具)
- 标签: `#jewelryasmr #luxuryjewelry #mythicaljewelry`

**类型 2: 测试结果 (25%)**

- 30 秒
- 录屏: 打开 mythrealms-shop.vercel.app/guardian-quiz 做测试
- 用 CapCut 加速录屏到 2 倍速
- 结尾戏剧性揭晓结果 + 截图
- 字幕: "I got the Nine-Tailed Fox 🦊 Which guardian are YOU?"
- 标签: `#guardianquiz #chinesemythology #whichbeastareyou`

**类型 3: 神话故事 (25%)**

- 45-60 秒
- 用 TikTok 文字工具 + 图片做幻灯片风格
- 节奏: 一张神话图 (3秒) → 文字说明 (2秒) → 产品图 (3秒) → 循环
- 内容: "2000 years ago, China's oldest book described a fox with nine tails..."
- 不要说话，用文字 + 音乐 (避免 accent 问题)

### 3.3 发布 SOP (标准操作流程)

**每次发布前:**
1. 确认 VPN 连接正常，IP 没变
2. 打开 `whatismyipaddress.com` 确认 IP 地址
3. 浏览 For You 页面 5 分钟 (模拟正常用户)

**发布时:**
1. 点 + 号 → 上传视频
2. 不要用 TikTok 相机拍 (用系统相机拍好再上传，画质更好)
3. 加字幕: 用 TikTok 内置文字工具，不要用其他 App 生成的字幕
4. 加音乐: 从 TikTok 音效库选，不要用外部导入的
5. 写描述: 2-3 句自然英文 + 3-5 个标签
6. 点 **"Post"**

**发布后:**
1. 自己完整看一遍视频
2. 给自己点赞 + 收藏
3. 回复前 3 条评论 (如果有)
4. 不要再改描述、不要删视频、不要设私密

### 3.4 发布频率建议

| 频率 | 适合 | 效果 |
|:--:|------|------|
| 1 条/天 | 有时间 | 最佳，算法最友好 |
| 3-4 条/周 | 常规 | 稳定增长 |
| 1-2 条/周 | 忙碌 | 维持不降权 |

> TikTok 偏爱稳定频率(每天同时段发) > 数量(一天发 3 条然后一周不发)。

## 第四部分: 防封完全指南

### 4.1 TikTok 封号触发因素

| 行为 | 风险 |
|------|:--:|
| IP 频繁切换 | 🔴 极高 |
| 新号第一天发视频 | 🔴 极高 |
| 视频带其他平台水印 (抖音/快手) | 🔴 极高 |
| 用虚拟号注册 | 🔴 极高 |
| 搬运他人视频 | 🟠 高 |
| 每条视频都带链接 | 🟠 高 |
| AI 语音朗读纯文字 | 🟠 高 |
| 视频画质模糊 | 🟡 中 |
| 视频时长 <5 秒 | 🟡 中 |

### 4.2 如果被限流了怎么办

**限流表现:** 播放量卡在 0-50，不增长。

1. 不要删视频、不要设私密、不要改描述
2. 不要连发新视频 "测试" 是否恢复
3. 像正常用户一样浏览 3-5 天 (不发布任何内容)
4. 3-5 天后发一条 15 秒生活化视频 (不要带任何商业元素)
5. 如果这条过 100 播放 → 恢复了
6. 如果仍然卡 0-50 → 这个号废了，换号

### 4.3 换号注意事项

不要用同一个手机、同一个 IP、同一个邮箱注册新号。这三个因素只要有一个重复，新号也会被关联封禁。
